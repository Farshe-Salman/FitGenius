# FitGenius

**FitGenius** is a **full-stack AI-powered fitness application** that provides **personalized workout and nutrition recommendations**. Built with **Spring Boot, React, Microservices, Keycloak, Docker, Kubernetes, and AWS**, it demonstrates a modern scalable architecture for real-world applications.

---

## Features
- Personalized AI-generated workout plans
- Nutrition guidance based on user goals
- Microservices architecture for modularity
- Authentication and authorization with Keycloak
- Frontend built with React
- Scalable deployment with Docker, Kubernetes, and AWS
